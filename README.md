# Rule-Based AI Chatbot 🤖

## Description
A simple Rule-Based AI Chatbot built in Python using if-else statements. The chatbot responds to predefined user inputs, handles greetings and exit commands, and runs continuously until the user exits.

## Features
- Greets users
- Responds to predefined questions
- Help command
- Exit command
- Continuous conversation loop
- Built using Python and if-else logic

## Technologies Used
- Python 3

## How to Run

1. Install Python 3.
2. Clone this repository or download the source code.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python chatbot.py
```

## Sample Commands

- hi
- hello
- how are you
- what is your name
- what is ai
- who created you
- help
- thank you
